# Superstore Global Sales Analysis

## Project Overview
This project analyzes the Global Superstore dataset using SQL and Excel to identify sales trends, profit patterns, and business insights across different regions, categories, and customer segments.

## Tools Used
- **SQL** (DB Browser for SQLite) — Data querying and analysis
- **Microsoft Excel** — Dashboard creation and data visualization

## Dataset
- **Source:** Kaggle — Global Superstore Dataset
- **Records:** 51,290 orders
- **Time Period:** 2011–2014
- **Markets:** Africa, APAC, Canada, EMEA, EU, LATAM, US

## Dashboard Features
- **KPIs:** Total Sales, Total Profit, Total Orders, Average Discount
- **Charts:** Sales by Region, Sales by Category, Yearly Sales Trend, Top 10 Sub-Categories, Profit by Region & Category

## Key Insights
- Technology category generated the highest profit among all categories.
- Central region had the highest total sales globally.
- Furniture had lower profit margins despite high sales volume.
- Consumer segment contributes the most to total sales.
- Standard Class is the most commonly used shipping mode.
